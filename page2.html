Intorduction of the Decision-making Support System

Decision support system is a system developed on the basis of the deepening of management information system application concept. DSS is a management information system that solves unstructured problems and serves high-level decision-making. According to its functions, it can be divided into special DSS, DSS tool and DSS generator

dss generators are divided into

  • database
  • Model library
  • Method library
  • Knowledge base
  • Session unit

the purpose of the decision support system:

In organizations, decision support systems (DSS) analyze and synthesize large amounts of data to aid decision making. Using this information, it can generate reports that can predict revenue, sales, or manage inventory. By integrating multiple variables, DSS can produce many different results based on a company's previous data and current inputs.

Advantage:

Decision support systems help make more informed decisions.

At the same time, the decision support system can be used to generate easy-to-digest reports for customers that can be adjusted to user specifications.
